Download NeuralNetwork zip file and unzip it
Use eclipse for running the program
Open eclipse
Choose File -> Import --> Existing project
Navigate to the file where NeuralNetwork is saved
Select NeuralNetwork folder and click on ok.
Once the project is loaded in Eclipse 
Select Run --> Run Configuration --> Java Application --> Arguments
Enter the input url, output url, learning rate and iterations separated by space
eg: http://www.hlt.utdallas.edu/~yangl/cs6375/homework/hw3/train.dat http://www.hlt.utdallas.edu/~yangl/cs6375/homework/hw3/test.dat 0.001 2000
Click on run.



If run button is disabled
Choose Run --> Run in eclipse. Project will give compilation error.

Enter the input and output urls separated by space
